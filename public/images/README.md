# Images Directory

This directory contains image assets for the landing page.

## Structure

- `hero/` - Hero section images
- `features/` - Feature section images
- `sections/` - Images for other sections (DeepDive, ValueProps, UseCases, etc.)

## Usage

Images in this directory can be referenced in components using paths starting with `/images/`.

Example:
```tsx
<img src="/images/hero/hero-image.jpg" alt="Hero image" />
```

## Supported Formats

- JPG/JPEG
- PNG
- WebP (recommended for better performance)
- SVG (for icons/logos)
